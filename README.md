# Portfolio

The repository for my portfolio website.

Stack used:
- VueJS (composition API)
- Typescript
- TailWind
- Scss

## Deployment

1. ```npm run build```
2. (Optionally test the build ```firebase emulators:start```)
3. ```firebase deploy```
